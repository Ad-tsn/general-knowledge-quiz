# general-knowledge-quiz

A small culture quiz web app built with Vue3 and Supabase.

The app fetches **5 quiz questions** from an external API, lets you play a game, and stores results in Supabase and display the **last 5 games** in the history page.

## Features

- Fetches 5 questions per game from an external quiz API
- Timer for each question
- Score calculation and end screen
- Game history: displays the last 5 games

## Tech Stack

- Vue 3 + Vite
- JavaScript
- Supabase
